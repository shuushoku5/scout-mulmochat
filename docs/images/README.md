# Screenshots

このフォルダにスクリーンショットを配置してください。

## 必要な画像

1. `screenshot_status.png` — 「状態を見せて」の結果（ステータスパネル）
2. `screenshot_action.png` — 「0.2m前進して」の結果（Before/After）
3. `screenshot_decision.png` — 異常時の選択肢フォーム
4. `screenshot_routes.png` — ルート一覧パネル

## 撮り方

ブラウザで http://localhost:7860 を開き、各コマンドを実行した画面をスクリーンショットしてください。
